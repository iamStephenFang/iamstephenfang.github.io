新版本的 Vue CLI 提供了 Dart Sass 与 node-sass 两种类型的 Sass 配置，在sass-loader的implementation中两者有所不同。

原因在于在 sass-loader v7 中，prependData选项名是 “data”，已经不适用。
默认情况下 sass 选项会同时对 sass 和 scss 语法同时生效，因为 scss 语法在内部也是由 sass-loader 处理的，但是在配置 data 选项的时候，scss 语法会要求语句结尾必须有分号，sass 则要求必须没有分号。在这种情况下，我们可以使用 scss 选项，对 scss 语法进行单独配置。实例代码来自官方文档。
